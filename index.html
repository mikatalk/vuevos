This project describes how to make a fried egg across various rendering Web APIs.

It uses Vue framework to navigate from one concept to another. Each fried egg (and its code) is encapsulated and available in its own component.

  • HTML/CSS

  • SVG Path

  • SVG Filters

  • Canvas 2D

  • WEBGL 2D

  • WEBGL 3D
